Creating Your Own Website

Introduction

This page goes over how to build your own webpage, from my own perspective. Not so much a teaching page as a personal reminder on how to do this, IMOW, as a quick reference.

Before you Start

Create a folder for your web projects. In the web project, you will have different folders for sites on there. In each site folder, you should have:

  1. a folder for images
  2. a folder for scripts (JS files go here)
  3. a folder for styles (CSS files go here)
  4. an HTML file(s) to access the webpage

You can create HTML, CSS, and JS files through Visual Code Studio. In our first tests, we used "index.html", "style.css" in styles folder, and "main.js" in scripts folder.
